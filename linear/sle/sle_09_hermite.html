Следующую задачу можно рассматривать как обратную к предыдущей, и хотя ее при подходящей трактовке можно решить исключительно на уровне манипуляций с многочленами, можно также ограничиться составлением и последующим решением системы линейных уравнений, что при необходимости гораздо проще переносится на уровень «железа» или может быть ускорено методами приближенных вычислений.

Задача. Решить задачу интерполяции по Эрмиту, то есть восстановления многочлена наименьшей возможной степени по набору его значений и значений его производных в наборе точек.

На вход подается набор строк, состоящих из разделенных пробелами целых чисел:\[c\ \ v_0\ \ v_1\ \ \ldots \ \ v_n\]Каждая такая строка налагает на многочлен \(f\) следующие ограничения:\[f(c)=v_0, \quad f'(c)=v_1, \quad \ldots \quad f^{(n)}(c)=v_n.\]Вывод программы должен состоять из разделенных пробелами коэффициентов многочлена \(f\) (от свободного члена к старшему коэффициенту).

Примеры ниже показывают, что частными случаями данной задачи являются интерполяция по Лагранжу и вычисление коэффициентов исходя из разложения по Тейлору в точке.

  • Первый пример показывает восстановление многочлена \((x+1)^2\) по его значениям в точках \(0\), \(1\), \(2\).
  • Второй пример показывает, что \(1 + (x-2) + (x-2)^2 + (x-2)^3 + (x-2)^4 = 11 - 23x +19x^2 -7x^3 + x^4\).
