В порядке разминки предлагается написать основу нужного нам класса (который мы далее будем всюду называть Matrix). Для начала оснастите его методами для прибавления одной строки к другой и для перестановки двух строк местами, а затем используйте их для приведения матрицы к строчной эшелонированной форме (ступенчатому виду).

Напомним, матрица называется строчной эшелонированной, если

  • все нулевые строки расположены ниже ненулевых строк;
  • ведущий элемент каждой строки (то есть первый ненулевой) расположен правее ведущего элемента предыдущей строки.

Обратите внимание, что ответ не является однозначно определенным и зависит от порядка операций.

Совет дня. С одной стороны, удобно реализовывать указанные операции in-place, чтобы не копировать на каждом шаге всю матрицу в новый объект, а менять данный (если нужно сохранить старый, сделайте копию перед преобразованиями). Но с другой стороны, теперь функция ничего полезного не возвращает. Можно использовать появившуюся свободу для реализации method chaining, например:

class Matrix:
    def __init__(self, entries):
        # инициализация матрицы по элементам
        pass

    def __str__(self):
        # возвращает требуемое для вывода представление матрицы,
        # чтобы можно было вызывать команду print(mat) для матрицы mat
        pass

    def row_swap(self, i, j):
        # какой-то код для перестановки строк i и j
        return self

    def row_add(self, i, j, x):
        # какой-то код для прибавления j-ой строки к i-ой с коэффициентом x
        return self

Наличие return self позволяет удобно применять последовательность преобразований, что очень полезно, кроме прочего, для тестирования и проработки примеров, например:

mat = Matrix([[1, 2, 3], [4, 5, 6]])
mat.row_add(0, 1, 1).row_swap(0, 1)

Теперь в mat записана матрица

[[4 5 6]
 [5 7 9]]

Задача. На вход подается матрица с целыми элементами, на выходе требуется какая-нибудь ее строчная эшелонированная форма, полученная перестановками строк и прибавлениями одной строки к другой (без домножения строки на множитель).
