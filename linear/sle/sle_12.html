Для реализации рассмотренного метода решения СЛУ в целых числах полезно дополнить класс Matrix несколькими новыми методами, например, для перестановки столбцов.

В рассмотренном алгоритме требуется также домножать на матрицу, отличающуюся от единичной только в одном \(2 \times 2\)-блоке. Конечно, можно конструировать нужную матрицу, но домножать на нее затратно — хотя меняются только две строки или два столбца. Конечно, можно требуемое изменение строк прописать вручную, но удобнее разработать метод для замены части матрицы. В Python для этой цели можно использовать «магический метод» __setitem__(self, key, value). Кроме очевидного случая, когда key = (i, j) это пара индексов, а value это скаляр, в котором в позицию \((i, j)\) записывается значение value, можно, используя пару слайсов в качестве ключа и матрицу в качестве значения, заменять сразу много элементов. Тогда, например, можно будет писать выражения вроде a[1:3, :] = c @ a[1:3, :] — если c это \(2 \times 2\)-матрица, то подматрица, составленная из второй и третьй строк матрицы a, будет заменена на ее произведение с матрицей c.
