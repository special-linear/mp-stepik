Реализуйте схему Горнера и используйте ее последовательно для представления данного многочлена \[f = a_0 + a_1x + a_2x^2 + \ldots + a_nx^n\] в виде \[f = b_0 + b_1(x-c) + b_2(x-c)^2 + \ldots + b_n(x-c)^n.\]На вход подается две строки: первая содержит разделенные пробелами целые числа \(a_0, a_1, \ldots, a_n\), вторая содержит целое число \(c\). Вывод программы должен состоять из одной строки, содержащей разделенные пробелами числа \(b_0, b_1, \ldots, b_n\).
